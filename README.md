# AtCoder

AtCoderの学習記録用リポジトリです。
Pythonを用いたアルゴリズムの実装と、基礎文法の習得を目的に取り組んでいます。

## 学習状況・進捗
現在は、基礎固めを中心に以下のコンテンツに取り組んでいます。

### 1. AtCoder Beginners Selection (ABS)
- AtCoderの過去問から選抜された重要問題（精選10問）の実装。
- 基礎的なアルゴリズムと、標準入出力の処理の確認。

### 2. APG4b (Python入門)
- AtCoder Programming Guide for beginners (EX20まで完了)。
- Pythonの基本文法を体系的に学習。

### 3. その他
- **Boot camp for Beginners**: 基礎問題の演習。Medium問題を1日3問以上解く。Easy問題は気分転換やウォームアップで解く。
- **Contest**: Beginners Contestに毎週参加、Heuristic Contestにも参加、その他参加できるContestには全参加。
- **Past Contest**: 苦手なアルゴリズムの問題演習。

## 使用言語
- Python 3.13.7

## 目標
- 基本的なアルゴリズムを適切な計算量で実装できる力をつける。
- Pythonの標準ライブラリ（collections, itertools, mathなど）を使いこなす。
- 当面はコンテストに参加し続け、入緑をまずは目指す。
